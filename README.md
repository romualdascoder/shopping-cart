## Getting Started

### Prerequisites

You must have Node.js and npm installed on your machine.

### Installation

1. Clone the repo

   ```sh
   $ git clone https://github.com/romualdascoder/shopping-cart
   ```

2. Install NPM packages

   ```sh
   npm install
   ```

### Start a project

1. Start

   ```sh
   npm start
   ```

2. Open this link

   ```sh
   http://localhost:3000
   ```
